# Inmobiliaria

## Descripción

Aplicación web desarrollada con JavaScript que permite visualizar propiedades en venta y arriendo mediante la generación dinámica de contenido. La información de los inmuebles se almacena en arreglos de objetos y se muestra en tarjetas creadas dinámicamente mediante manipulación del DOM.

## Funcionalidades

- Visualización de propiedades en venta.
- Visualización de propiedades en arriendo.
- Renderizado dinámico de tarjetas.
- Información detallada de cada inmueble.
- Indicadores visuales para mascotas y fumadores.
- Diseño responsivo con Bootstrap.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap
- Font Awesome

## Objetivos de aprendizaje

- Manipulación del DOM.
- Uso de arreglos y objetos.
- Renderizado dinámico de contenido.
- Uso de estructuras condicionales.
- Desarrollo de interfaces web dinámicas.
